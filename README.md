## Installation

Use the package manager [npm](https://www.npmjs.com/) to install MovieApp.


To run client-side:
vite/master branch
```bash
cd frontend
npm install
npm run dev
```
create-react-app branch
```bash
cd frontend
npm install
npm start
```

To run server-side:
```bash
cd backend
npm install
npm start
```
